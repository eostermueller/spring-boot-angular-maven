# Getting Started
Build the pr0ject like this:
```./mvnw clean install```
...which will create this:
```backend/target/backend-0.0.1-SNAPSHOT.jar```
Then to run: 
```cd backend```
...and either this:
```../mvnw spring-boot:run```
or this:
```java -jar backend-0.0.1-SNAPSHOT.jar```
